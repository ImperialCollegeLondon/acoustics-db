Welcome to the SAFE Acoustics Database webpage

This website maintains a database of automatic acoustic samples recorded at the SAFE Project. It is primarily used as an API and catalogue for recorded files.

Acoustics website

Our Acoustics website provides a user-friendly interface to our acoustics files and to look at the different habitats and species sampled.
